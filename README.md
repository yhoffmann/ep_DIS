# ep-DIS (A test for DIS-like diffractive cross-section resolver)

## Requirements 
Cuba integration Library (available at https://feynarts.de/cuba/)  
GNU Scientific Library

## How to compile 
 
To compile you need to simply run  
```
cd path/to/build/fldr
make
```
and to run  
```
./ep.exe
```
